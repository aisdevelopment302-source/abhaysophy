# Abhaysophy - Design Brief

## Brand
**Name:** Abhaysophy  
**Tagline:** "Abhay bhai is always right, sometimes."  
**Audience:** Teens & Gen Z (India)  
**Vibe:** Ironic philosophy, meme-aware humor, streetwear meets intellect

## Aesthetic Direction

**Theme:** Dark, minimal, refined with subtle irreverence  
**Inspiration:** Bluorng's clean luxury aesthetic + meme culture undertones

### Typography
- **Display Font:** Clash Display or similar bold geometric sans
- **Body Font:** Satoshi or similar clean modern sans
- **Accent:** Occasional serif for "philosophy" moments

### Color Palette
- **Primary:** Off-black (#0D0D0D)
- **Background:** Near-black (#111111)  
- **Accent:** Electric yellow (#FFE600) or lime (#B8FF00)
- **Text:** Off-white (#F5F5F5)
- **Muted:** Gray (#666666)

### Design Principles
1. **Clean grid layouts** — products as heroes
2. **Bold typography** — statements, not whispers
3. **Ironic copy** — clever, self-aware, never cringe
4. **Generous whitespace** — let it breathe
5. **Subtle animations** — hover states, page transitions
6. **Dark theme default** — premium, Gen Z preferred

## Pages Required

### 1. Home Page
- Hero with brand statement + featured product
- Product grid (latest drops)
- About/philosophy section with a witty quote
- Newsletter signup

### 2. Shop Page
- Filter by category (tees, hoodies, etc.)
- Product grid with quick view
- Sort options

### 3. Product Detail Page
- Large product images (gallery)
- Size selector
- Add to cart
- Product description with brand voice

### 4. Cart Page
- Line items with quantity control
- Subtotal
- Checkout button (mockup only)

### 5. About Page
- Brand story with irreverent tone
- The "Abhay bhai" philosophy explained
- Team/founder section

## Technical Requirements

- Next.js 15 App Router
- Tailwind CSS v4
- TypeScript
- Client-side cart state (React Context)
- Responsive (mobile-first)
- No external UI libraries (custom components)
- Google Fonts or Fontsource for typography
